# 资金信息管理系统差异版

《资金信息管理系统差异版》是针对企业、单位、个体经营户等管理资金和往来账款而开发的一款软件，可以设置账号权限管理不同的账户类型，记录资金的收入、支出，操作简便、查询功能简洁明了，是管理者的好帮手！   
系统包含以下功能：   
基本信息：账户类型、用户信息、公司信息、发生单位、职工信息、银行信息；   
业务管理：收入登记、支出登记、收支统计。  
 软件特点：   
 1.与excle表格交换数据；  
 2.业务操作简洁明了。   
 
# 如何运行本系统?

本系统基于宏达数据库信息管理开发平台开发，下载后，双击文件夹中"dbimp.exe"文件即可完整运行对应的信息管理系统。
